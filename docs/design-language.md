# Design Language

- **Style:** minimal, clean, and readable  
- **Background:** `#f9fafb`  
- **Text:** `#111827`  
- **Accent:** `#3b82f6` (soft blue)
- **Fonts:**  
  - Body → Inter  
  - Headings → Inter Tight or Satoshi  
  - Code → JetBrains Mono
- **Max width:** ~760px  
- **Spacing:** roomy, airy, calm  
- **Corners:** slightly rounded  
- **Shadows:** very subtle  
- **Animations:** none (static calmness)

The vibe should feel calm, confident, and human — not like a template.  
Focus on typography and whitespace over visual clutter.
